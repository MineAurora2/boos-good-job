# 求职雷达 / BOSS 投递高保真原型

这是一个无需构建工具、无需网络依赖的可点击高保真演示。直接双击 `index.html` 或在浏览器地址栏打开即可。

## 交互演示

- 左侧工作区导航（移动端为底部五项导航）切换总览、自动化、分析、记录、设置。
- 顶栏连接菜单可演示在线、加载中、离线、需授权状态；主题按钮切换深浅色。
- 自动化列表的「•••」可暂停运行中的投递计划，已暂停计划可继续；暂停需要危险操作确认。
- 分析页支持投递趋势、转化漏斗、地区/薪资/岗位/行业三个 tabs 与时间范围筛选。
- 记录页支持关键词搜索、状态筛选、移动端筛选 sheet、详情抽屉和 Escape 关闭/焦点恢复。
- 设置页输入内容后进入 dirty 状态，离开工作区会询问确认，保存按钮恢复干净状态。

## 验证

在仓库根目录运行：

```powershell
node --check reports/dashboard-ui-demo/app.js
node --test --test-isolation=none tests/test_dashboard_ui_demo.js
```

页面为静态演示，所有数据均为内嵌样例，不会修改生产 dashboard、后端或真实配置。
